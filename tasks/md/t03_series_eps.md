# Task t03_series_eps

Claim: For every eps > 0, there exists N such that for all n >= N,

  sum_{k=n}^{2n} (1/k) < eps.

Prove or disprove the claim. Your answer must include a verdict and a very short
proof sketch.

Required format:
- Verdict: true/false
- Proof sketch: 3-6 sentences
