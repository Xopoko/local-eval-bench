# Task t04_ineq_border

Claim: For all real x with 0 <= x <= 1, we have

  x^2 < x.

Prove or disprove the claim. You must explicitly check the boundary cases.
Your answer must include a verdict and a very short proof sketch.

Required format:
- Verdict: true/false
- Proof sketch: 3-6 sentences
